# Customer_Behaviour_Analysis-
Data Analysis project Showcasing Customer behaviour analysis using Python, SQL, Power BI

📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering actionable insights through dashboards and reports. It covers data loading, cleaning, exploratory data analysis (EDA), SQL querying, and visualization.

The goal of this project is to extract meaningful insights and present them in a clear, business-friendly format.

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI (Dashboard & visualization)
Jupyter Notebook / VS Code
🔄 Project Workflow
1. Data Loading
Imported dataset using Python (Pandas)
Performed initial inspection (shape, columns, data types)
2. Data Cleaning
Handled missing values
Removed duplicates
Standardized column formats
Fixed inconsistent data entries
3. Exploratory Data Analysis (EDA)
Univariate and bivariate analysis
Identified trends, patterns, and outliers
Created visualizations for better understanding
4. SQL Analysis
Loaded cleaned data into databases:
PostgreSQL
MySQL
SQL Server
Wrote queries to:
Aggregate data
Perform joins
Extract key business insights
5. Data Visualization (Power BI)
Built interactive dashboard
Included KPIs and key metrics
Enabled filtering for dynamic insights
